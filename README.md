# Ejercicio-8-html

Este ejercicio demuestra cómo controlar el comportamiento del navegador al interactuar con hipervínculos externos hacia plataformas de blogs.

## Estructura de la Página
El contenido se divide en dos secciones principales para comparar el comportamiento de los enlaces:

1. **Apertura en otra pestaña**: Enlaces configurados con `target="_blank"` para mantener la navegación actual abierta.
2. **Apertura en el mismo navegador**: Enlaces con el comportamiento por defecto (sin atributo target), que cargan el nuevo sitio en la ventana activa.

## Tecnologías y Etiquetas
* `<a>`: Hipervínculos con rutas absolutas.
* `<ul>` / `<li>`: Listas no ordenadas para organizar los directorios de blogs.
* `<hr>`: Separador temático entre ambas secciones.

## Sitios Enlazados
* Blogger
* WordPress
* Medium
